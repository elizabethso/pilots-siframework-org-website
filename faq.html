Transitions of Care (ToC) initiative

What transitions of care user stories are considered for ToC pilots?

From the ToC Use Case, the following health information exchange scenarios are considered:

  1. Discharge Summary
  2. Discharge Instructions
  3. Clinical Summary including Consultation Request
  4. Clinical Summary

What functionality does the ToC Reference Implementation (RI) provide?

ToC reference implementation is a programmatically accessible Reference Library modeling ToC data format(s), and a tool which will utilize the reference library to compose and validate ToC Information Package using ToC data from a source format.
